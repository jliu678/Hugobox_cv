Expected Value of Kinetic State Probability Vector

This document explains how to calculate the expected value of the kinetic state probability vector $\pi_{ng}$ , where this expectation is taken twice: first conditional on a latent variable $z_n$ , and then averaged over the possible values of $z_n$ for a given cell.

The main expression we’re evaluating is:

$$ \mathbb{E}_{q_\phi(z_n|u_n,s_n)} \left[ \mathbb{E}_{q_\phi(\pi_{ng}|z_n)} [\pi_{ng}] \right] \tag{1} $$

This represents VeloVI’s average belief about the kinetic state probabilities ($\pi_{ng}$ ) for gene $g$ in cell $n$ , considering all possible latent representations ($z_n$ ) that could explain the observed data ($u_n, s_n$ ).

Understanding the Components

  • $\pi_{ng}$ : The vector of probabilities for the 4 kinetic states: $(\pi_{ng,1}, \pi_{ng,2}, \pi_{ng,3}, \pi_{ng,4})$
  • $q_\phi(\pi_{ng}|z_n)$ : The approximate posterior distribution for $\pi_{ng}$ given a specific latent variable $z_n$ . This is a Dirichlet distribution with concentration parameters $\alpha_{q,ng}(z_n) = (\alpha_{q,ng,1}(z_n), \ldots, \alpha_{q,ng,K}(z_n))$
  • $q_\phi(z_n|u_n,s_n)$ : The approximate posterior distribution for the latent variable $z_n$ given the observed counts. This is a Gaussian distribution $\mathcal{N}(\mu_{z_n}, \Sigma_{z_n})$

Step 1: Calculate the Inner Expectation

The inner expectation $\mathbb{E}_{q_\phi(\pi_{ng}|z_n)} [\pi_{ng}]$ is the expected value of a Dirichlet-distributed random variable.

If $\pi_{ng} \sim \text{Dirichlet}(\alpha_{q,ng,1}(z_n), \ldots, \alpha_{q,ng,K}(z_n))$ , then the expected value of its $i$ -th component is:

$$ \mathbb{E}_{q_\phi(\pi_{ng}|z_n)} [\pi_{ng,i}] = \frac{\alpha_{q,ng,i}(z_n)}{\sum_{j=1}^K \alpha_{q,ng,j}(z_n)} \tag{2} $$

The result of the inner expectation is a vector of expected probabilities:

$$ \bar{\pi}_{ng}(z_n) = \left( \frac{\alpha_{q,ng,1}(z_n)}{\sum_j \alpha_{q,ng,j}(z_n)}, \ldots, \frac{\alpha_{q,ng,K}(z_n)}{\sum_j \alpha_{q,ng,j}(z_n)} \right) \tag{3} $$

Numeric Example for Cell 1, Gene A

Let’s assume we sample $z_1^* = (0.6558, -0.3779)$ .

The encoder outputs Dirichlet concentration parameters for Gene A: $\alpha_{q,1A}(z_1^*) = (5.0, 0.1, 0.2, 0.1)$

Calculate the expected $\pi_{1A}$ for this $z_1^*$ :

  • Sum of concentrations: $5.0 + 0.1 + 0.2 + 0.1 = 5.4$
  • $\mathbb{E}[\pi_{1A,1}|z_1^*] = 5.0/5.4 \approx 0.9259$
  • $\mathbb{E}[\pi_{1A,2}|z_1^*] = 0.1/5.4 \approx 0.0185$
  • $\mathbb{E}[\pi_{1A,3}|z_1^*] = 0.2/5.4 \approx 0.0370$
  • $\mathbb{E}[\pi_{1A,4}|z_1^*] = 0.1/5.4 \approx 0.0185$

So $\bar{\pi}_{1A}(z_1^*) = (0.9259, 0.0185, 0.0370, 0.0185)$ .

Step 2: Calculate the Outer Expectation

Now we need to average $\bar{\pi}_{ng}(z_n)$ over the distribution $q_\phi(z_n|u_n,s_n) = \mathcal{N}(\mu_{z_n}, \Sigma_{z_n})$ . Since $\bar{\pi}_{ng}(z_n)$ involves a neural network mapping, this expectation is intractable and we use Monte Carlo approximation.

Monte Carlo Approximation

For each sample $m \in \{1, \ldots, M\}$ :

  1. Draw random noise: $\epsilon_z^{(m)} \sim \mathcal{N}(0, I_d)$
  2. Calculate sample: $z_n^{*(m)} = \mu_{z_n} + \sqrt{\Sigma_{z_n}} \odot \epsilon_z^{(m)}$ (reparameterization trick)

Numeric Example Continued

Known parameters for Cell 1:

  • $\mu_{z_1} = (0.5, -0.3)$
  • $\log\Sigma_{z_1} = (-0.5, -0.5) \rightarrow \Sigma_{z_1} = \text{diag}(0.6065, 0.6065)$

Sample 1:

  • $\epsilon_z^{(1)} = (0.2, -0.1)$
  • $z_1^{*(1)} = (0.5, -0.3) + (0.7788, 0.7788) \odot (0.2, -0.1) = (0.6558, -0.3779)$

Sample 2:

  • $\epsilon_z^{(2)} = (-0.5, 0.3)$
  • $z_1^{*(2)} = (0.5, -0.3) + (0.7788, 0.7788) \odot (-0.5, 0.3) = (0.1106, -0.0663)$

For $z_1^{*(2)}$ , assume encoder outputs $\alpha_{q,1A}(z_1^{*(2)}) = (4.0, 0.3, 0.1, 0.1)$ :

  • Sum of concentrations: $4.0 + 0.3 + 0.1 + 0.1 = 4.5$
  • $\bar{\pi}_{1A}^{(2)} \approx (0.8889, 0.0667, 0.0222, 0.0222)$

Final average (with M=2):

$$ \mathbb{E}_{q_\phi(z_n|u_n,s_n)} [\bar{\pi}_{ng}(z_n)] \approx \frac{1}{M} \sum_{m=1}^M \bar{\pi}_{ng}^{(m)} \tag{4} $$
  • Component 1: $(0.9259 + 0.8889)/2 = 0.9074$
  • Component 2: $(0.0185 + 0.0667)/2 = 0.0426$
  • Component 3: $(0.0370 + 0.0222)/2 = 0.0296$
  • Component 4: $(0.0185 + 0.0222)/2 = 0.0204$

Final result: $\approx (0.9074, 0.0426, 0.0296, 0.0204)$

Expected Latent Time Calculation

Now we calculate the average latent time $t_{ng}$ for gene $g$ in cell $n$ , accounting for uncertainty in both the cell’s latent representation and the kinetic state.

The expression is:

$$ \mathbb{E}_{q_\phi(z_n|u_n,s_n)} \left[ \mathbb{E}_{q_\phi(\pi_{ng}|z_n)} [t^{(k_{ng})}_{ng}] \right] \tag{5} $$

where $t^{(k_{ng})}_{ng}$ represents the latent time for gene $g$ in cell $n$ if it’s in kinetic state $k_{ng}$ .

Derive the Inner Expectation for Time

The inner expectation over kinetic states is:

$$ \mathbb{E}_{q_\phi(\pi_{ng}|z_n)} [t^{(k_{ng})}_{ng}] = \sum_{k=1}^K P(k_{ng} = k | z_n) \cdot t_{ng}^{(k)}(z_n) \tag{6} $$

Where:

  • $P(k_{ng} = k | z_n) = \frac{\alpha_{q,ng,k}(z_n)}{\sum_{j=1}^K \alpha_{q,ng,j}(z_n)}$
  • $t_{ng}^{(k)}(z_n) = \rho_{ng}^{(k)}(z_n) \times t_g^s$ (decoder output times switching time)

In our expression $\mathbb{E}_{q_\phi(\pi_{ng}|z_n)} [t^{(k_{ng})}_{ng}]$ :

  • The random variable: The random variable is $t^{(k_{ng})}_{ng}$ . This variable’s value depends on the discrete kinetic state $k_{ng}$ .

  • The Possible Values of the Random Variable

    • The kinetic state $k_{ng}$ can take on $K$ discrete values (1, 2, 3, 4 for VeloVI’s four states).
    • For each specific state $k$ , the corresponding value of our random variable is $t_{ng}^{(k)}(z_n)$ .
    • This notation $t_{ng}^{(k)}(z_n)$ explicitly shows that the latent time is specific to state $k$ and is determined by $z_n$ .

  • The Probability Distribution

    • The expectation is taken with respect to $q_\phi(\pi_{ng}|z_n)$ .
    • This distribution tells us the probabilities of being in each kinetic state $k$ .
    • Specifically, $\pi_{ng}$ is a vector of probabilities for the states, and $q_\phi(\pi_{ng}|z_n)$ is a Dirichlet distribution that governs these probabilities.

We know that $k_{ng} \sim \text{Categorical}(\pi_{ng})$ . This means, if we knew the exact value of $\pi_{ng}$ , then $P(k_{ng}=k|\pi_{ng}) = \pi_{ng,k}$ .

However, $\pi_{ng}$ itself is a random variable distributed according to $q_\phi(\pi_{ng}|z_n)$ .

So, to get the marginal probability $P(k_{ng}=k|z_n)$ , we need to integrate over $\pi_{ng}$ :

$$ P(k_{ng}=k|z_n) = \int P(k_{ng}=k|\pi_{ng}) q_\phi(\pi_{ng}|z_n) d\pi_{ng} \tag{6.1} $$ $$ P(k_{ng}=k|z_n) = \int \pi_{ng,k} q_\phi(\pi_{ng}|z_n) d\pi_{ng} \tag{6.2} $$

This integral is simply the definition of the expected value of the $k$ -th component of $\pi_{ng}$ under the distribution $q_\phi(\pi_{ng}|z_n)$ :

$$ P(k_{ng}=k|z_n) = \mathbb{E}_{q_\phi(\pi_{ng}|z_n)} [\pi_{ng,k}] \tag{6.3} $$

For a Dirichlet distribution $\text{Dir}(\alpha_{q,ng,1}(z_n), \ldots, \alpha_{q,ng,K}(z_n))$ , this expectation has a closed form:

$$ \mathbb{E}_{q_\phi(\pi_{ng}|z_n)} [\pi_{ng,k}] = \frac{\alpha_{q,ng,k}(z_n)}{\sum_{j=1}^K \alpha_{q,ng,j}(z_n)} \tag{6.4} $$

So, $P(k_{ng}=k|z_n)$ is precisely this fraction.

Therefore:

$$ \mathbb{E}_{q_\phi(\pi_{ng}|z_n)} [t^{(k_{ng})}_{ng}] = \sum_{k=1}^K \left( t_{ng}^{(k)}(z_n) \cdot P(k_{ng}=k|z_n) \right) \tag{6} $$

Substituting equation (4) into equation (6):

$$ \mathbb{E}_{q_\phi(\pi_{ng}|z_n)} [t^{(k_{ng})}_{ng}] = \sum_{k=1}^K \left( t_{ng}^{(k)}(z_n) \cdot \frac{\alpha_{q,ng,k}(z_n)}{\sum_{j=1}^K \alpha_{q,ng,j}(z_n)} \right) \tag{6.5} $$

This is exactly the equation we use in practice, explicitly showing that the expectation is a weighted sum where:

  • Each possible latent time $t_{ng}^{(k)}(z_n)$ is weighted by
  • The probability of its corresponding state $k$ occurring, as determined by the approximate posterior $q_\phi(\pi_{ng}|z_n)$ .

Numeric Example for Expected Time

This expression asks for the average latent time $t_{ng}$ for gene g in cell n, taking into account both:

  • The uncertainty in the cell’s underlying latent representation $z_n$
  • The uncertainty in which kinetic state $k_{ng}$ gene g is in

Similar to the previous expectation, this involves an intractable integral, so we will use Monte Carlo sampling for approximation.

The expression is:

$$\mathbb{E}_{q_\phi(z_n|u_n,s_n)} \left[\mathbb{E}_{q_\phi(\pi_{ng}|z_n)} \left[t(k_{ng})_{ng}\right]\right] \tag{1}$$

Where $t(k_{ng})_{ng}$ represents the latent time for gene g in cell n if it’s in kinetic state $k_{ng}$ . Note that this latent time $t_{ng}$ is itself a function of $z_n$ and the specific kinetic state k.

Let’s break down the calculation in steps with a numeric example for Cell 1 and Gene A.

What We Assume We Have (after VeloVI training)

Observed Data for Cell 1, Gene A: $u_{1A} = 0.8, s_{1A} = 0.2$

Learned Biophysical Parameters (Œ∏) for Gene A:

  • Switching Time $(t_A^s) = 8.0$
  • (Other Œ∏ like Œ±,Œ≤,Œ≥ are not directly used in this specific calculation of expected time, but they would be used to calculate expected RNA levels from this time)

Learned Neural Network Parameters (œÜ) for Encoder and Decoder:

Encoder outputs for $z_1$ (given $u_1, s_1$ *):* Defines $q_\phi(z_1|u_1,s_1) = \mathcal{N}(\mu_{z_1}, \Sigma_{z_1})$

  • $\mu_{z_1} = (0.5, -0.3)$
  • $\Sigma_{z_1} = \text{diag}(e^{-0.5}, e^{-0.5}) \approx \text{diag}(0.6065, 0.6065)$ (for $z_n$ as 2D)

Encoder outputs for $\pi_{1A}$ (given $z_1$ *):* Defines $q_\phi(\pi_{1A}|z_1) = \text{Dirichlet}(\alpha_{q,1A}(z_1))$

Decoder outputs for $\rho_{1A}^{(k)}$ (given $z_1$ *):* For each state k, the decoder outputs a scaled latent time $\rho_{1A}^{(k)}$ , where $t_{1A}^{(k)} = \rho_{1A}^{(k)} \times t_A^s$ .

Step 1: Calculate the Inner Expectation

Recall the Inner Expectation

$$\mathbb{E}_{q_\phi(\pi_{ng}|z_n)} \left[t(k_{ng})_{ng}\right] \tag{2}$$

This inner expectation is over the kinetic states $k_{ng}$ , weighted by their probabilities. The latent time $t(k_{ng})_{ng}$ is specific to the state $k_{ng}$ . We can write this expectation as a sum:

$$\mathbb{E}_{q_\phi(\pi_{ng}|z_n)} \left[t(k_{ng})_{ng}\right] = \sum_{k=1}^K P(k_{ng} = k|z_n) \cdot t_{ng}^{(k)}(z_n) \tag{3}$$

Where:

  • K is the number of kinetic states (4 in VeloVI)
  • $P(k_{ng} = k|z_n)$ is the probability that gene g in cell n is in state k, given $z_n$ . This is the k-th component of the expected $\pi_{ng}$ vector from the Dirichlet distribution:
$$P(k_{ng} = k|z_n) = \frac{\alpha_{q,ng,k}(z_n)}{\sum_{j=1}^K \alpha_{q,ng,j}(z_n)} \tag{4}$$
  • $t_{ng}^{(k)}(z_n)$ is the latent time for gene g in cell n if it is in state k. This is a deterministic output of the decoder, given $z_n$ : $t_{ng}^{(k)}(z_n) = \text{Decoder}_k(z_n) \times t_g^s = \rho_{ng}^{(k)}(z_n) \times t_g^s$ .

The result of this inner expectation will be a single scalar value, let’s call it $\bar{t}_{ng}(z_n)$ , which is still a function of $z_n$ .

Numeric Example for Cell 1, Gene A

Here we take Cell 1, Gene A for a Numeric Example: assume we have a sample $z_1^* = (0.6558, -0.3779)$ (we’ll show how to get this in Step 2).

  1. From $z_1^*$ **, get Dirichlet concentration parameters:**

    • Assume encoder outputs: $\alpha_{q,1A}(z_1^*) = (5.0, 0.1, 0.2, 0.1)$
    • Sum of concentrations = 5.4

  2. Calculate $P(k_{1A} = k|z_1^*)$ for each state:

    • $P(k_{1A} = 1|z_1^*) = 5.0/5.4 \approx 0.9259$
    • $P(k_{1A} = 2|z_1^*) = 0.1/5.4 \approx 0.0185$
    • $P(k_{1A} = 3|z_1^*) = 0.2/5.4 \approx 0.0370$
    • $P(k_{1A} = 4|z_1^*) = 0.1/5.4 \approx 0.0185$

  3. From $z_1^*$ **, get latent times** $t_{1A}^{(k)}(z_1^*)$ for each state: (These are specific outputs from the decoder for this $z_1^*$ )

    • Assume decoder outputs $\rho_{1A}^{(1)}(z_1^*) = 0.5$ (for Induction) $t_{1A}^{(1)}(z_1^*) = 0.5 \times t_A^s = 0.5 \times 8.0 = 4.0$
    • Assume decoder outputs $\rho_{1A}^{(2)}(z_1^*) = 0.3$ (for Repression) $t_{1A}^{(2)}(z_1^*) = 0.3 \times t_A^s = 0.3 \times 8.0 = 2.4$
    • Assume decoder outputs $\rho_{1A}^{(3)}(z_1^*) = 0.8$ (for Steady-State Induction) $t_{1A}^{(3)}(z_1^*) = 0.8 \times t_A^s = 0.8 \times 8.0 = 6.4$
    • Assume decoder outputs $\rho_{1A}^{(4)}(z_1^*) = 0.1$ (for Steady-State Repression) $t_{1A}^{(4)}(z_1^*) = 0.1 \times t_A^s = 0.1 \times 8.0 = 0.8$

  4. Calculate the weighted sum $\bar{t}_{1A}(z_1^*)$ **:**

$$\bar{t}_{1A}(z_1^*) = (0.9259 \times 4.0) + (0.0185 \times 2.4) + (0.0370 \times 6.4) + (0.0185 \times 0.8) \tag{5}$$ $$\bar{t}_{1A}(z_1^*) = 3.7036 + 0.0444 + 0.2368 + 0.0148 \tag{6}$$ $$\bar{t}_{1A}(z_1^*) \approx 3.9996 \text{ (This is a scalar value for this specific } z_1^*) \tag{7}$$

Step 2: Monte Carlo Approximation Calculates the Outer Expectation

$$\mathbb{E}_{q_\phi(z_n|u_n,s_n)} \left[\bar{t}_{ng}(z_n)\right] \tag{8}$$

Now we need to average this $\bar{t}_{ng}(z_n)$ value over the distribution of $z_n$ . Since $\bar{t}_{ng}(z_n)$ is a complex, non-linear function of $z_n$ (involving neural network outputs), we use Monte Carlo approximation.

  1. Draw M samples of $z_n^*$ from $q_\phi(z_n|u_n,s_n)$ **:**

    Recall $q_\phi(z_1|u_1,s_1) = \mathcal{N}(\mu_{z_1}, \Sigma_{z_1})$ with $\mu_{z_1} = (0.5, -0.3)$ and $\Sigma_{z_1} = \text{diag}(0.6065, 0.6065)$ .

  2. For each sample $m \in \{1, ..., M\}$ **:**

    • Draw a random noise vector $\varepsilon_z^{(m)} \sim \mathcal{N}(0, I_d)$
    • Calculate $z_n^{*(m)} = \mu_{z_n} + \sqrt{\Sigma_{z_n}} \odot \varepsilon_z^{(m)}$ (This is the reparameterization trick)

Just two samples for brevity, usually M=1000+:

Sample 1:

  • $\varepsilon_z^{(1)} = (0.2, -0.1)$
  • $z_1^{*(1)} = (0.5, -0.3) + (\sqrt{0.6065}, \sqrt{0.6065}) \odot (0.2, -0.1) = (0.6558, -0.3779)$

Sample 2:

  • $\varepsilon_z^{(2)} = (-0.5, 0.3)$
  • $z_1^{*(2)} = (0.5, -0.3) + (\sqrt{0.6065}, \sqrt{0.6065}) \odot (-0.5, 0.3) = (0.1106, -0.0663)$

  1. For each sampled $z_n^{*(m)}$ **, calculate** $\bar{t}_{ng}(z_n^{*(m)})$ **:**

    Follow the steps for the inner expectation (from above) for each $z_n^{*(m)}$ .

For $z_1^{*(1)} = (0.6558, -0.3779)$ *:* We already calculated: $\bar{t}_{1A}^{(1)} \approx 3.9996$

For $z_1^{*(2)} = (0.1106, -0.0663)$ *:*

  • Assume encoder outputs $\alpha_{q,1A}(z_1^{*(2)}) = (4.0, 0.3, 0.1, 0.1)$ (differs from first sample). Sum = 4.5
  • $P(k_{1A} = 1|z_1^{*(2)}) = 4.0/4.5 \approx 0.8889$
  • $P(k_{1A} = 2|z_1^{*(2)}) = 0.3/4.5 \approx 0.0667$
  • $P(k_{1A} = 3|z_1^{*(2)}) = 0.1/4.5 \approx 0.0222$
  • $P(k_{1A} = 4|z_1^{*(2)}) = 0.1/4.5 \approx 0.0222$

Assume decoder outputs (different for this $z_1^{*(2)}$ ):

  • $\rho_{1A}^{(1)}(z_1^{*(2)}) = 0.4 \rightarrow t_{1A}^{(1)}(z_1^{*(2)}) = 0.4 \times 8.0 = 3.2$
  • $\rho_{1A}^{(2)}(z_1^{*(2)}) = 0.2 \rightarrow t_{1A}^{(2)}(z_1^{*(2)}) = 0.2 \times 8.0 = 1.6$
  • $\rho_{1A}^{(3)}(z_1^{*(2)}) = 0.7 \rightarrow t_{1A}^{(3)}(z_1^{*(2)}) = 0.7 \times 8.0 = 5.6$
  • $\rho_{1A}^{(4)}(z_1^{*(2)}) = 0.05 \rightarrow t_{1A}^{(4)}(z_1^{*(2)}) = 0.05 \times 8.0 = 0.4$

Calculate weighted sum $\bar{t}_{1A}(z_1^{*(2)})$ for this sample:

$$\bar{t}_{1A}(z_1^{*(2)}) = (0.8889 \times 3.2) + (0.0667 \times 1.6) + (0.0222 \times 5.6) + (0.0222 \times 0.4) \tag{9}$$ $$\bar{t}_{1A}(z_1^{*(2)}) = 2.8445 + 0.1067 + 0.1243 + 0.0089 \tag{10}$$ $$\bar{t}_{1A}(z_1^{*(2)}) \approx 3.0844 \tag{11}$$
  1. Average the results:

The overall expectation is approximated by the average of these M scalar values:

$$\mathbb{E}_{q_\phi(z_n|u_n,s_n)} [\bar{t}_{ng}(z_n)] \approx \frac{1}{M} \sum_{m=1}^M \bar{t}_{ng}^{(m)} \tag{12}$$

For our example with M=2:

Average latent time for Gene A in Cell 1:

$$\frac{3.9996 + 3.0844}{2} = 3.542 \tag{13}$$

This final scalar value (e.g., 3.542) represents VeloVI’s average estimated latent time for Gene A in Cell 1, taking into account all the uncertainties modeled by the variational distributions.

Calculate Expected Velocity for a sampled $z_n$

Note: Posterior Velocity Distribution, i.e. the full posterior predictive velocity distribution, is just the distribution of all the expected velocity values for the sample $z_n^*$ (drawn from $q_\phi(z_n|u_n, s_n)$ ).

$$\mathbb{E}_{q_\phi(\pi_{ng}|z_n)} \left[v^{(g)}(t(k_{ng})_{ng}, k_{ng})\right] \tag{1}$$

This term represents the expected velocity for gene g in cell n, given a specific latent variable $z_n$ , averaging over the uncertainty in its kinetic state $k_{ng}$ .

We will use the same logic as the previous expectation for latent time, as the structure is identical: we are calculating the expectation of a quantity ($v^{(g)}$ ) that depends on a discrete random variable ($k_{ng}$ ), whose probabilities are determined by the Dirichlet distribution $q_\phi(\pi_{ng}|z_n)$ .

1. Recall the General Expectation for a Discrete Variable

If X is a discrete random variable that can take on values $x_1, x_2, \ldots, x_K$ with corresponding probabilities $P(X=x_1), P(X=x_2), \ldots, P(X=x_K)$ , then:

$$\mathbb{E}[X] = \sum_{i=1}^K x_i \cdot P(X=x_i) \tag{2}$$

2. Applying to Our Velocity Case

The Random Variable: Our random variable is $v^{(g)}(t(k_{ng})_{ng}, k_{ng})$ . Its value depends on the specific kinetic state $k_{ng}$ .

The Possible Values: For each state $k \in \{1,2,3,4\}$ , the value of the velocity is $v^{(g)}(t_{ng}^{(k)}(z_n), k)$ . This velocity depends on the latent time $t_{ng}^{(k)}(z_n)$ (which is a function of $z_n$ and k) and the state k itself.

The Probabilities: The probability of being in state k is $P(k_{ng} = k|z_n)$ . As derived previously, this is given by the expected value of the k-th component of $\pi_{ng}$ under the Dirichlet distribution:

$$P(k_{ng} = k|z_n) = \mathbb{E}_{q_\phi(\pi_{ng}|z_n)}[\pi_{ng,k}] = \frac{\alpha_{q,ng,k}(z_n)}{\sum_{j=1}^K \alpha_{q,ng,j}(z_n)} \tag{3}$$

Putting it together, the inner expectation for velocity becomes:

$$\mathbb{E}_{q_\phi(\pi_{ng}|z_n)} \left[v^{(g)}(t(k_{ng})_{ng}, k_{ng})\right] = \sum_{k=1}^K P(k_{ng} = k|z_n) \cdot v^{(g)}(t_{ng}^{(k)}(z_n), k) \tag{4}$$

This will result in a single scalar value for each gene and cell, given a specific $z_n$ .

Numeric Example for Cell 1, Gene A

Let’s use the same example inputs from our previous discussions:

Assumed Sampled $z_1^*$ : $(0.6558, -0.3779)$ (This comes from the first step in the “posterior predictive velocity distribution” process).

Learned Biophysical Parameters (Œ∏) for Gene A:

  • $\alpha_A = 1.2$ (Transcription rate in Induction)
  • $\beta_A = 0.2$ (Splicing rate)
  • $\gamma_A = 0.1$ (Degradation rate)
  • $t_A^s = 8.0$ (Switching time)

Encoder Output for $\pi_{1A}$ (given $z_1^*$ **):**

  • $\alpha_{q,1A}(z_1^*) = (5.0, 0.1, 0.2, 0.1)$
  • Sum = 5.4

This gives us the state probabilities $P(k_{1A} = k|z_1^*)$ :

  • $P(k_{1A} = 1|z_1^*) = 5.0/5.4 \approx 0.9259$
  • $P(k_{1A} = 2|z_1^*) = 0.1/5.4 \approx 0.0185$
  • $P(k_{1A} = 3|z_1^*) = 0.2/5.4 \approx 0.0370$
  • $P(k_{1A} = 4|z_1^*) = 0.1/5.4 \approx 0.0185$

Decoder Output for $\rho_{1A}^{(k)}$ (given $z_1^*$ **) and corresponding latent times** $t_{1A}^{(k)}$ **:**

  • $\rho_{1A}^{(1)} = 0.5 \Rightarrow t_{1A}^{(1)} = 0.5 \times 8.0 = 4.0$
  • $\rho_{1A}^{(2)} = 0.3 \Rightarrow t_{1A}^{(2)} = 0.3 \times 8.0 = 2.4$
  • $\rho_{1A}^{(3)} = 0.8 \Rightarrow t_{1A}^{(3)} = 0.8 \times 8.0 = 6.4$
  • $\rho_{1A}^{(4)} = 0.1 \Rightarrow t_{1A}^{(4)} = 0.1 \times 8.0 = 0.8$

The Velocity Formula:

$$v^{(g)}(t^{(k)}, k) = \beta_g \bar{u}^{(g)}(t^{(k)}, k) - \gamma_g \bar{s}^{(g)}(t^{(k)}, k) \tag{5}$$

We need the expected unspliced ($\bar{u}$ ) and spliced ($\bar{s}$ ) counts for each state at its corresponding latent time. These are given by the kinetic equations in VeloVI.

Calculations for each state k and its corresponding $t_{1A}^{(k)}$ :

State 1 (Induction, k=1): t=4.0

$$\bar{u}^{(A)}(4.0, 1) = \frac{\alpha_A}{\beta_A}(1 - e^{-\beta_A t}) = \frac{1.2}{0.2}(1 - e^{-0.2 \times 4.0}) = 6(1 - e^{-0.8}) \approx 6(1 - 0.4493) = 3.3042 \tag{6}$$ $$\bar{s}^{(A)}(4.0, 1) = \frac{\alpha_A}{\gamma_A}\left(1 - \frac{\gamma_A}{\gamma_A - \beta_A}e^{-\beta_A t} + \frac{\beta_A}{\gamma_A - \beta_A}e^{-\gamma_A t}\right) \tag{7}$$ $$= \frac{1.2}{0.1}\left(1 - \frac{0.1}{0.1 - 0.2}e^{-0.8} + \frac{0.2}{0.1 - 0.2}e^{-0.4}\right) = 12(1 + e^{-0.8} - 2e^{-0.4}) \tag{8}$$ $$\approx 12(1 + 0.4493 - 2 \times 0.6703) = 12(1.4493 - 1.3406) \approx 1.3044 \tag{9}$$ $$v^{(A)}(4.0, 1) = \beta_A \bar{u}^{(A)}(4.0, 1) - \gamma_A \bar{s}^{(A)}(4.0, 1) = (0.2 \times 3.3042) - (0.1 \times 1.3044) = 0.66084 - 0.13044 = 0.5304 \tag{10}$$

State 2 (Repression, k=2): t=2.4

(For repression, we need the values at $t_A^s = 8.0$ (end of induction) as initial conditions for decay.)

$$\bar{u}^{(A)}(t_A^s, 1) = 6(1 - e^{-0.2 \times 8.0}) \approx 6(1 - 0.2019) = 4.7886 \tag{11}$$ $$\bar{s}^{(A)}(t_A^s, 1) = 12(1 + e^{-1.6} - 2e^{-0.8}) \approx 12(1 + 0.2019 - 2 \times 0.4493) = 3.6396 \tag{12}$$ $$\bar{u}^{(A)}(2.4, 2) = \bar{u}^{(A)}(t_A^s, 1)e^{-\beta_A t_{1A}^{(2)}} = 4.7886 \times e^{-0.2 \times 2.4} \approx 4.7886 \times 0.6188 = 2.9634 \tag{13}$$ $$\bar{s}^{(A)}(2.4, 2) = \bar{s}^{(A)}(t_A^s, 1)e^{-\gamma_A t_{1A}^{(2)}} + \frac{\beta_A \bar{u}^{(A)}(t_A^s, 1)}{\gamma_A - \beta_A}(e^{-\beta_A t_{1A}^{(2)}} - e^{-\gamma_A t_{1A}^{(2)}}) \tag{14}$$ $$= 3.6396e^{-0.1 \times 2.4} + \frac{0.2 \times 4.7886}{0.1 - 0.2}(e^{-0.2 \times 2.4} - e^{-0.1 \times 2.4}) \tag{15}$$ $$\approx 3.6396 \times 0.7866 + (-9.5772)(0.6188 - 0.7866) \approx 2.8617 + 1.6072 = 4.4689 \tag{16}$$ $$v^{(A)}(2.4, 2) = (0.2 \times 2.9634) - (0.1 \times 4.4689) = 0.59268 - 0.44689 = 0.1458 \tag{17}$$

State 3 (Steady-State Induction, k=3): t=6.4

$$\bar{u}^{(A)}(6.4, 3) = \alpha_A/\beta_A = 1.2/0.2 = 6.0 \tag{18}$$ $$\bar{s}^{(A)}(6.4, 3) = \alpha_A/\gamma_A = 1.2/0.1 = 12.0 \tag{19}$$ $$v^{(A)}(6.4, 3) = \beta_A \bar{u}^{(A)}(6.4, 3) - \gamma_A \bar{s}^{(A)}(6.4, 3) = (0.2 \times 6.0) - (0.1 \times 12.0) = 1.2 - 1.2 = 0.0 \tag{20}$$

State 4 (Steady-State Repression, k=4): t=0.8

$$\bar{u}^{(A)}(0.8, 4) = 0.0 \tag{21}$$ $$\bar{s}^{(A)}(0.8, 4) = 0.0 \tag{22}$$ $$v^{(A)}(0.8, 4) = \beta_A \bar{u}^{(A)}(0.8, 4) - \gamma_A \bar{s}^{(A)}(0.8, 4) = (0.2 \times 0.0) - (0.1 \times 0.0) = 0.0 \tag{23}$$

Calculate the weighted sum:

Now, we multiply each calculated velocity by its corresponding state probability $P(k_{1A} = k|z_1^*)$ and sum them up:

$$\mathbb{E}_{q_\phi(\pi_{1A}|z_1^*)} \left[v^{(A)}(t(k_{1A})_{1A}, k_{1A})\right] = \tag{24}$$ $$(0.9259 \times 0.5304) + (0.0185 \times 0.1458) + (0.0370 \times 0.0) + (0.0185 \times 0.0) \tag{25}$$ $$= 0.4900 + 0.0027 + 0.0 + 0.0 \approx 0.4927 \tag{26}$$

Get Posterior Predictive Velocity Distribution

The value 0.4927 is the expected velocity for Gene A in Cell 1, given the specific sampled latent variable $z_1^* = (0.6558, -0.3779)$ .

This value is one sample of the posterior predictive velocity. To get the full posterior predictive velocity distribution, you would repeat this entire calculation for many different samples of $z_n^*$ (drawn from $q_\phi(z_n|u_n, s_n)$ ) and then analyze the distribution of these resulting expected velocity values.
